3. Adjectives: Intensity (Comparison)
Adjectives use suffixes to show degrees of comparison. Grammarly
Comparative: Adding -er to compare two things (e.g., fast → faster). EF United States
Superlative: Adding -est to indicate the highest degree in a group (e.g., fast → fastest). Albert.io
Spelling Shifts: If an adjective ends in -y, it changes to -i before the suffix (happy → happier). Lingo Apps


5.Adverbs are almost always formed by adding a suffix to a base word (usually an adjective). Wordy
Derivational Suffixes:
-ly: The most common way to turn an adjective into an adverb (e.g., quick → quickly, quiet → quietly). GINGER
-wise: Added to nouns to show direction or manner (e.g., clockwise, lengthwise, otherwise). Scribd
-ward(s): Added to indicate direction (e.g., backward, homeward, upwards). Scribd
Spelling Rules for "-ly":
If the base ends in -y, it changes to -i (e.g., happy → happily). My English Class
If it ends in -le, drop the -e and add -y (e.g., gentle → gently). My English Class
If it ends in -ic, add -ally (e.g., tragic → tragically). My English Class

1. Decompose the Event (MMS Mapping)
Before feeding the Markov chain, break the generated Topic (Who + What + When + Where + Why) into individual sentence goals (p. 10).
Topic 1 (Who1): Start the chain with the Primary Actor (p. 6).
Topic 2 (Why1): Generate a causal sentence linked to the Reason/Cause using an Emotional State or Mental Intent verb (pp. 5, 10).
Topic 3 (What3): Generate the Affected Outcome or Reaction sentence (p. 6).
2. Constraint the Chain by Verb Type
The Markov chain must be "filtered" by the Verb/Action Type determined in the event roll (pp. 1, 10):
If Movement/Manual: Use the Actor -> Enacted -> Action Outcome path (p. 5).
If Cognition/Perception: Use the Actor -> Enacted -> Mystery (Info-Related) path (p. 6).
If Relational/Linking: Use the Actor -> Enacted -> Roll Affected path (p. 6).
